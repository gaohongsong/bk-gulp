## django工程下的gulp实践

### step1. 拷贝gulpfile.js和package.json到工程根目录

### step2. 执行 npm install -g gulp

### step3. 执行 npm install

### step4. 执行 gulp <help|build|rebuild|zip|minifyjs|cleancss|minifyhtml|jshint|minifyjs6|sass>
